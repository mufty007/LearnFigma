Tips & Tricks

Some Tricks

Draw a Star with 8 Points and set Ratio/Radius to 34%.⁣
Add Outside Stroke and set Joins to Bevel.⁣
Go to “Edit Mode” and Select the inner radius points.
Increase the Corner Radius to create the Cog Shape.⁣
Change Star’s Fill Color, so that it matches with the Stroke.⁣
Use the Oval Shape tool and draw a circle in the center.⁣
⁣⁣⁣⁣⁣⁣Use Oval, Rectangle, or Pen tool to draw a Shape.⁣⁣⁣⁣⁣⁣⁣
Add Inner Shadow as per required border-side⁣⁣⁣⁣⁣⁣.⁣
Top - X: 0 | Y: 3 | Blur: 0 |Spread: 0⁣⁣⁣⁣⁣⁣
Bottom - X: 0 | Y: -3 | Blur: 0 |Spread: 0⁣⁣⁣⁣⁣⁣⁣
Left - X: 3 | Y: 0 | Blur: 0 |Spread: 0⁣⁣⁣⁣⁣⁣⁣
Right - X: -3 | Y: 0 | Blur: 0 |Spread: 0 ⁣⁣⁣⁣⁣
⁣⁣⁣⁣⁣⁣Use Oval, Rectangle, or Pen tool to draw a Shape.⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣
Add a thin dark-colored Stroke to the Shape and set its position to “Inside”.⁣⁣⁣⁣⁣
Add Light & Dark Drop Shadows - Light Shadow Fill : #FFFFFF⁣⁣⁣⁣ X: 4 Y: 4 Blur: 0 Spread: 0⁣⁣⁣⁣⁣⁣
Dark Shadow Fill : #312F35⁣⁣⁣⁣
X: 10 Y: 10 Blur: 0 Spread: 0⁣⁣⁣⁣⁣⁣⁣
