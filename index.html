How to create a settings Icons

Steps

  • Draw a Star with 8 Points and set Ratio/Radius to 34%.
  • Add Outside Stroke and set Joins to Bevel.⁣
  • Go to “Edit Mode” and Select the inner radius points. Increase the Corner Radius to create the Cog Shape.
  • Change Star’s Fill Color, so that it matches with the Stroke.⁣
  • Use the Oval Shape tool and draw a circle in the center.⁣

How to create a Retro Style Icons

Steps

  • DUse Oval, Rectangle, or Pen tool to draw a Shape.⁣
  • Add a thin dark-colored Stroke to the Shape and set its position to “Inside”.⁣⁣⁣⁣⁣
  • Add Light & Dark Drop Shadows - Light Shadow Fill : #FFFFFF⁣⁣⁣⁣ X: 4 Y: 4 Blur: 0 Spread: 0 || Dark Shadow Fill : #312F35⁣⁣⁣⁣ X: 10 Y: 10 Blur: 0 Spread: 0⁣⁣⁣⁣
  • Add “Inner Shadow” and set its options -X: 8 Blur: 0 Y: 8 Spread: 0⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣⁣
  • Draw other elements and repeat the same steps or Copy / Paste the Layer Style to re-use.⁣

How to create a single Side borders

Steps

  • Use Oval, Rectangle, or Pen tool to draw a Shape.⁣⁣⁣⁣⁣⁣
  • Add Inner Shadow as per required border-side⁣⁣⁣⁣⁣⁣:
